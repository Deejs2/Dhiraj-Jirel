Khalti ePayment Gateway Integration

July 26, 2024 by Dhiraj Jirel

This project demonstrates the process of integrating Khalti's latest e-Payment Checkout platform into a PHP application. The integration allows merchants to accept payments on their website using Khalti, providing a smooth and secure payment experience.

Features

  • Simple integration with Khalti's e-Payment platform.
  • Secure payment processing.
  • Detailed response handling and error management.
  • Easy to extend and customize for your own needs.

Prerequisites

  • XAMPP
  • PHP 7.4 or higher
  • cURL extension enabled
  • Khalti merchant account (optional: use test secret key)

Getting Started

  1. Clone the repository: 
    git clone https://github.com/Deejs2/khalti-ePayment-gateway.git
  2. Navigate to the project directory: 
    cd khalti-ePayment-gateway
  3. Configure the project by updating the paths in payment-request.php: 
    
                            "return_url" => "http://localhost/khalti-ePayment-gateway/payment-response.php",
                            "website_url" => "http://localhost/khalti-ePayment-gateway/"
  4. Host the project on your local server or any PHP compatible hosting service.
